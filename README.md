# 🎯 Jogo de Sorteio de Números

Projeto desenvolvido em JavaScript puro onde o usuário escolhe até 50 números entre 0 e 99, e o sistema realiza um sorteio automático de 20 números, exibindo os acertos.

---

## 🚀 Funcionalidades

* Seleção de até 50 números
* Sorteio de 20 números aleatórios (sem repetição)
* Exibição dos números acertados
* Feedback em tempo real da quantidade de números escolhidos
* Reset do jogo

---

## 🧠 Conceitos aplicados

* Manipulação de DOM
* Eventos (`click`, `DOMContentLoaded`)
* Arrays e métodos avançados:

  * `map`
  * `filter`
  * `some`
  * `findIndex`
* Modularização com ES Modules (`import/export`)
* Controle de estado
* Uso de `DocumentFragment` para otimização de performance

## 📌 Regras do jogo

* O usuário pode escolher até 50 números
* O sistema sorteia 20 números aleatórios
* Os números acertados são destacados na tela
* Após o sorteio, é necessário resetar o jogo para jogar novamente
